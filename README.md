💰 Financial Dashboard Web App
📌 Overview

This project is a modern, responsive financial dashboard designed to track income and expenses with a clean and intuitive user interface. It provides users with real-time insights through charts, summaries, and transaction management.

The application follows a fintech-style UI design, focusing on usability, performance, and scalability.

🚀 Features
📊 Dashboard Analytics
Real-time income & expense summary cards
Interactive charts for data visualization
Financial insights for better decision-making
➕ Transaction Management
Add income and expense entries
View transactions in a structured table
Edit and delete transactions
🔍 Search & Filter
Search transactions by name
Filter by type (income/expense)
🧑‍💼 Role-Based Access Control (RBAC)
Different user roles with controlled access
Restricted actions based on permissions
📤 Export Data
Export transactions as CSV file
🌙 Dark Mode
Toggle between light and dark themes
📱 Fully Responsive
Works smoothly on:
Mobile phones 📱
Tablets 📟
Laptops 💻
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling (Glassmorphism + Responsive Design)
JavaScript (Vanilla JS) – Functionality & State Handling
Chart.js – Data visualization

🧠 Approach & Design Decisions

🔄 State Management
Centralized transaction array

Mobile-first optimization for better accessibility
